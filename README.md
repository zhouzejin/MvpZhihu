# 知道

一款使用MVP架构的MaterialDesign风格的阅读类APP.

## 简介

「知道」- 每天3次，每次7分钟

项目基于Dagger + RxJava + Retrofit实现；  
采用MVP架构，UI参考HotBitmapGG的RxZhiHu开源项目；  
灵感来自知乎日报客户端，喜欢的朋友们可以star一下，谢谢啦~

## 功能介绍

1.日报列表  
2.日报详情  
3.主题日报  
4.日报专栏  
5.热门新闻  
6.日报评论  
7.日报分享
